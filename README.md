# CP8 Cli
CP8 is a command line tool which aids in interacting with Github.

CP8 helps you open draft pull requests, suggests branch names and opens your CI for a branch.

## Installation

```bash
gem install cp8_cli
```

## Usage

```bash
cp8 help
```
